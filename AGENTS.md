# local-memo-app

localstorageにデータを保存する、シンプルなメモアプリケーションです。

## 機能要件

- メモの追加
  - メモには、タイトル、内容が含まれる。
  - 編集すると、自動的に保存される。
- メモの編集
  - メモの編集には、MarkdownモードとPlaintextモードがある。
  - Plaintextモードでは、特に書式は無く、純粋にテキストを編集できる。（textareaでただ編集するイメージ）
  - Markdown モードでは、Markdownの書式を記述すると、リアルタイムで見た目が反映される。
- メモの削除
  - メモを削除する際には、確認ダイアログを表示する
- メモの一覧表示
  - メモのタイトルと内容の一部を表示する。
  - メモの内容が長い場合は、省略して表示する。
- メモの検索
  - メモのタイトルと内容を対象に検索できる。
- メモの並び替え
  - 作成日時、更新日時、タイトルで並び替えができる。

## 非機能要件

- レスポンシブデザイン
  - PC、タブレット、スマートフォンで快適に利用できる。
- パフォーマンス
  - メモの数が多くなっても、可能な限り快適に動作する。
- セキュリティ
  - XSS攻撃に対する対策を講じる。ただし、そもそもlocalstorageにしか保存しないので、外部からの攻撃ベクトルは限定的。

## 技術要件

- フロントエンドフレームワーク: vite
- スタイル： tailwindcss
- デプロイ： GitHub Pages
